The Airport Express Line is a fictional line mainly based on Singapore and New York. The route has been created using Routebuilder, by Nickgoh45. Upgrades are done in-house by SMRTO Studios.

The support for the legacy C651/A/B/C/D has been deprecated. Download Version 1.0 to use it for the Legacy Trains.
